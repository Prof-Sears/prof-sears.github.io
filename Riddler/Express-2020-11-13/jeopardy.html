Jeopardy! Simulation

Systematic: Low-to-High

This simulation runs through the Jeopardy! clues from lowest value to highest value. It tests every possible location of the Daily Double and averages the winnings.

Systematic: High-to-Low

This simulation runs through the Jeopardy! clues from highest value to lowest value. It tests every possible location of the Daily Double and averages the winnings.

Random Searching

This simulation answers the clues in random order with random locations for the Daily Double. It runs many trials of the game and averages the winnings.
