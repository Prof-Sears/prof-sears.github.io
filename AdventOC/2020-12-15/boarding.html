Plane Boarding Simulation

This page is to solve the programming problem from Advent of Code - Day 5.

From the problem statement, it looks like we can turn the seat specifications into seat number using binary arithmetic. The ticket description divides the plane into zones, with the first character dividing the plane into halves, the second divides each half into halves, and so on. This sounds excactly what the binary digits do to a range of numbers. Further, combining the seat and row numbers by taking the row number and multiplying by 1000 in binary and adding a three bit number fits the ticket pattern.

To test this, input the example tickets to the box below.

The code I used for the conversion is the following:


        const binString = ticket.replace(/F/g,'0').replace(/B/g,'1').replace(/L/g,'0').replace(/R/g,'1');
        const seatID = parseInt(binString,2);

After some testing, it looks like this code works! Now, on to the main portion.

To finish the problem, we'll do the following:

  1. Fetch the data from the Advent of Code Website.
  2. Parse the text file and convert the known tickets to seat ids.
  3. Sort for the largest seat ID.
  4. Search for the missing seat ID.
